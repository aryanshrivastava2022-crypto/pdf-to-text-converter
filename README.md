# PDF to Text Converter
A simple Python application that extracts text from PDF files and saves it into a text file.

# Features
- Extract text from PDF documents
- Save extracted text as TXT files
- Easy-to-use command-line interface
- Supports multi-page PDFs

# Requirements
- Python 3.x
- PyPDF2

# Installation
Install the required library:
pip install PyPDF2

# How to Run
1. Place your PDF file in the project folder.
2. Open a terminal in the project directory.
3. Run:
pdf_to_text_converter.py
4. Enter the PDF filename.
5. The extracted text will be saved as:
output.txt

# Sample Output
